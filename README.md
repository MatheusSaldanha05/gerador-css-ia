# ⚡ Gerador de CSS com IA

Uma aplicação web que utiliza Inteligência Artificial para gerar código HTML + CSS a partir de descrições em linguagem natural, com preview em tempo real.


---

## 💡 Como funciona

1. O usuário digita uma descrição (ex: *"bola azul pulando"*)
2. A aplicação envia essa descrição para uma API de IA
3. A IA gera código HTML + CSS
4. O código é exibido na tela
5. Um preview é renderizado automaticamente

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- API de Inteligência Artificial (Groq)
